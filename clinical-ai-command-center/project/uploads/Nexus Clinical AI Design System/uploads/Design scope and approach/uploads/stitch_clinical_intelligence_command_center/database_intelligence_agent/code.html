Agent Summary

The query retrieved patient risk trends for the past 30 days. The data indicates a steady increase in high-risk classifications, peaking mid-month. Processed 14,203 records across 2 schemas.

Generated SQL 
SELECT 
    DATE_TRUNC('day', admission_date) AS trend_date,
    COUNT(*) AS high_risk_patient_count
FROM 
    patients_core
WHERE 
    risk_score >= 0.8 
    AND admission_date >= CURRENT_DATE - INTERVAL '30 days'
GROUP BY 
    1
ORDER BY 
    trend_date ASC;
