Primary action button — flat, bordered, 6px radius; use for any clickable command.

```jsx
<Button variant="primary" icon="add">New Session</Button>
<Button variant="outline" size="sm">Review Draft</Button>
<Button variant="danger" icon="warning">Flag High Risk</Button>
```

Variants: `primary` (clinical blue fill), `secondary` (slate container), `outline` (blue hairline), `ghost` (text-only), `danger` (critical red). Sizes: `sm` (36px), `md` (44px). Add `icon`/`iconRight` for Material Symbols glyphs, `fullWidth` to fill the container.
